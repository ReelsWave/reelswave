# Autopilot Feature - Complete Implementation Plan for ReelsWave

## Codebase Summary

The project is a clean, small codebase with a clear separation:
- **Backend**: Express 5 server (ESM modules), 4 services (scriptGenerator, voiceGenerator, stockFetcher, videoAssembler), 2 route files (videos, payments), 1 auth middleware. Uses an in-memory job queue (Map + Array) for sequential video processing. `googleapis` is already a dependency (^171.4.0) but unused currently.
- **Frontend**: React 19 + Vite 7 + React Router 7, uses plain CSS (no Tailwind), has `lucide-react` for icons. Pages: Landing, Login, Signup, Dashboard, Create (5-step wizard). Dark premium theme with CSS variables in `index.css`.
- **Database**: Supabase with `profiles` table (id, plan, credits, ls_*) and `videos` table (id, user_id, title, topic, niche, tone, script, video_url, storage_path, status, hashtags, created_at).
- **Auth**: Supabase JWT validation via service-role key on the backend. Frontend uses anon key.

```javascript
import { google } from 'googleapis';
import crypto from 'crypto';
import dotenv from 'dotenv';
dotenv.config();

const router = express.Router();

// ===== YOUTUBE =====

// Stores ephemeral state -> userId mapping for OAuth CSRF protection
const oauthStateMap = new Map();

router.get('/youtube/connect', authMiddleware, (req, res) => { ... });
// - Create a google.auth.OAuth2 client
// - Generate a random `state` string, store userId against it in oauthStateMap
// - Build authUrl with scopes: ['https://www.googleapis.com/auth/youtube.upload',
//   'https://www.googleapis.com/auth/youtube.readonly']
// - Redirect user to authUrl

router.get('/youtube/callback', async (req, res) => { ... });
// - Validate `state` from oauthStateMap to get userId
// - Exchange `code` for tokens via oauth2Client.getToken(code)
// - Fetch channel info via youtube.channels.list({ part: 'snippet', mine: true })
// - Upsert into social_connections table
// - Redirect to frontend: `${FRONTEND_URL}/autopilot?connected=youtube`

// ===== TIKTOK =====

router.get('/tiktok/connect', authMiddleware, (req, res) => { ... });
// - Generate CSRF state, store in oauthStateMap
// - Build TikTok auth URL:
//   https://www.tiktok.com/v2/auth/authorize/?client_key=...&scope=video.publish,video.upload,user.info.basic
//   &response_type=code&redirect_uri=...&state=...
// - Redirect user

router.get('/tiktok/callback', async (req, res) => { ... });
// - Validate state
// - Exchange code for tokens via POST https://open.tiktokapis.com/v2/oauth/token/
//   body: { client_key, client_secret, code, grant_type: 'authorization_code', redirect_uri }
// - Fetch user info via GET https://open.tiktokapis.com/v2/user/info/
//   headers: { Authorization: 'Bearer <access_token>' }
//   query: { fields: 'open_id,display_name,avatar_url' }
// - Upsert into social_connections
// - Redirect to frontend

// ===== INSTAGRAM =====

router.get('/instagram/connect', authMiddleware, (req, res) => { ... });
// - Generate state
// - Build Meta auth URL:
//   https://www.facebook.com/v19.0/dialog/oauth?client_id=...
//   &redirect_uri=...&scope=instagram_basic,instagram_content_publish,pages_show_list,pages_read_engagement
//   &state=...&response_type=code
// - Redirect user

router.get('/instagram/callback', async (req, res) => { ... });
// - Validate state
// - Exchange code for short-lived token via POST https://graph.facebook.com/v19.0/oauth/access_token
// - Exchange short-lived token for 60-day long-lived token via GET
//   https://graph.facebook.com/v19.0/oauth/access_token?grant_type=fb_exchange_token&...
// - Find Instagram Business Account:
//   1. GET /me/accounts -> get Facebook Pages
//   2. For each page, GET /{page_id}?fields=instagram_business_account
// - Upsert into social_connections with the long-lived token
// - Redirect to frontend

// ===== SHARED ENDPOINTS =====

router.get('/connections', authMiddleware, async (req, res) => { ... });
// - Query social_connections for req.user.id
// - Return { connections: [...] } with tokens stripped (never send tokens to frontend)

router.delete('/disconnect/:platform', authMiddleware, async (req, res) => { ... });
// - Delete from social_connections where user_id and platform match
// - Also deactivate any schedules that target this platform exclusively

export default router;
```

```javascript
import { google } from 'googleapis';
import fs from 'fs';
import { getValidToken } from './tokenRefresher.js';

/**
 * Upload a video to YouTube Shorts
 * @param {Object} options
 * @param {string} options.userId - User ID for token lookup
 * @param {string} options.videoPath - Local path to the .mp4 file
 * @param {string} options.title - Video title (will append #Shorts)
 * @param {string} options.description - Video description
 * @param {string[]} options.tags - Hashtag array
 * @returns {Object} { platformVideoId, platformUrl }
 */
export async function uploadToYouTube({ userId, videoPath, title, description, tags }) {
    const accessToken = await getValidToken(userId, 'youtube');
    
    const oauth2Client = new google.auth.OAuth2();
    oauth2Client.setCredentials({ access_token: accessToken });
    
    const youtube = google.youtube({ version: 'v3', auth: oauth2Client });
    
    // Ensure #Shorts is in the title for YouTube Shorts classification
    const shortsTitle = title.includes('#Shorts') ? title : `${title} #Shorts`;
    
    const response = await youtube.videos.insert({
        part: ['snippet', 'status'],
        requestBody: {
            snippet: {
                title: shortsTitle.substring(0, 100), // YouTube title limit
                description: `${description}\n\n${tags.map(t => `#${t}`).join(' ')}`,
                tags: tags,
                categoryId: '22' // People & Blogs
            },
            status: {
                privacyStatus: 'public',
                selfDeclaredMadeForKids: false
            }
        },
        media: {
            body: fs.createReadStream(videoPath)
        }
    });
    
    return {
        platformVideoId: response.data.id,
        platformUrl: `https://youtube.com/shorts/${response.data.id}`
    };
}
```

### `Backend/services/tiktokUploader.js`

```javascript
import fs from 'fs';
import axios from 'axios';
import { getValidToken } from './tokenRefresher.js';

/**
 * Upload a video to TikTok via Direct Post (Content Posting API)
 * @param {Object} options
 * @returns {Object} { platformVideoId, platformUrl }
 */
export async function uploadToTikTok({ userId, videoPath, title, description, tags }) {
    const accessToken = await getValidToken(userId, 'tiktok');
    
    const videoSize = fs.statSync(videoPath).size;
    const caption = `${title}\n${tags.map(t => `#${t}`).join(' ')}`.substring(0, 2200);
    
    // Step 1: Initialize upload via Direct Post
    const initResponse = await axios.post(
        'https://open.tiktokapis.com/v2/post/publish/video/init/',
        {
            post_info: {
                title: caption,
                privacy_level: 'PUBLIC_TO_EVERYONE',
                disable_duet: false,
                disable_comment: false,
                disable_stitch: false
            },
            source_info: {
                source: 'FILE_UPLOAD',
                video_size: videoSize,
                chunk_size: videoSize,     // Single chunk for files < 64MB
                total_chunk_count: 1
            }
        },
        {
            headers: {
                'Authorization': `Bearer ${accessToken}`,
                'Content-Type': 'application/json; charset=UTF-8'
            }
        }
    );
    
    const { upload_url, publish_id } = initResponse.data.data;
    
    // Step 2: Upload the video binary
    const videoBuffer = fs.readFileSync(videoPath);
    await axios.put(upload_url, videoBuffer, {
        headers: {
            'Content-Type': 'video/mp4',
            'Content-Length': videoSize,
            'Content-Range': `bytes 0-${videoSize - 1}/${videoSize}`
        }
    });
    
    // Step 3: Check publish status (poll until complete or timeout)
    let attempts = 0;
    const maxAttempts = 30; // 5 minutes at 10s intervals
    while (attempts < maxAttempts) {
        await new Promise(r => setTimeout(r, 10000));
        const statusRes = await axios.post(
            'https://open.tiktokapis.com/v2/post/publish/status/fetch/',
            { publish_id },
            { headers: { 'Authorization': `Bearer ${accessToken}`, 'Content-Type': 'application/json' } }
        );
        
        const status = statusRes.data.data?.status;
        if (status === 'PUBLISH_COMPLETE') {
            return {
                platformVideoId: publish_id,
                platformUrl: null // TikTok does not return a direct URL in the API response
            };
        }
        if (status === 'FAILED') {
            throw new Error(`TikTok publish failed: ${statusRes.data.data?.fail_reason || 'unknown'}`);
        }
        attempts++;
    }
    
    throw new Error('TikTok upload timed out after 5 minutes');
}
```

### `Backend/services/instagramUploader.js`

```javascript
import axios from 'axios';
import { getValidToken } from './tokenRefresher.js';
import { supabase } from '../middleware/auth.js';

/**
 * Upload a Reel to Instagram via Graph API (container-based two-step publish)
 * Requires the video to be at a publicly accessible URL (Supabase storage)
 * @param {Object} options
 * @returns {Object} { platformVideoId, platformUrl }
 */
export async function uploadToInstagram({ userId, videoUrl, title, description, tags }) {
    const accessToken = await getValidToken(userId, 'instagram');
    
    // Get the Instagram Business Account ID from social_connections
    const { data: connection } = await supabase
        .from('social_connections')
        .select('platform_user_id')
        .eq('user_id', userId)
        .eq('platform', 'instagram')
        .single();
    
    if (!connection) throw new Error('Instagram account not connected');
    
    const igUserId = connection.platform_user_id;
    const caption = `${title}\n\n${tags.map(t => `#${t}`).join(' ')}`.substring(0, 2200);
    
    // Step 1: Create media container
    const containerRes = await axios.post(
        `https://graph.facebook.com/v19.0/${igUserId}/media`,
        {
            media_type: 'REELS',
            video_url: videoUrl,   // Must be a publicly accessible URL
            caption: caption
        },
        { params: { access_token: accessToken } }
    );
    
    const containerId = containerRes.data.id;
    
    // Step 2: Poll container status until ready
    let attempts = 0;
    const maxAttempts = 60; // 10 minutes at 10s intervals
    while (attempts < maxAttempts) {
        await new Promise(r => setTimeout(r, 10000));
        const statusRes = await axios.get(
            `https://graph.facebook.com/v19.0/${containerId}`,
            { params: { fields: 'status_code', access_token: accessToken } }
        );
        
        if (statusRes.data.status_code === 'FINISHED') break;
        if (statusRes.data.status_code === 'ERROR') {
            throw new Error('Instagram container processing failed');
        }
        attempts++;
    }
    
    // Step 3: Publish
    const publishRes = await axios.post(
        `https://graph.facebook.com/v19.0/${igUserId}/media_publish`,
        { creation_id: containerId },
        { params: { access_token: accessToken } }
    );
    
    return {
        platformVideoId: publishRes.data.id,
        platformUrl: `https://www.instagram.com/reel/${publishRes.data.id}/`
    };
}
```

### `Backend/services/scheduler.js`

This is the core autopilot engine. It initializes a `node-cron` job that runs every minute, finds schedules that are due, generates a unique topic angle, runs the video pipeline, and uploads to all connected platforms.

```javascript
import cron from 'node-cron';
import { v4 as uuidv4 } from 'uuid';
import path from 'path';
import fs from 'fs';
import { supabase } from '../middleware/auth.js';
import { generateScript } from './scriptGenerator.js';
import { generateVoiceover } from './voiceGenerator.js';
import { fetchStockFootage } from './stockFetcher.js';
import { assembleVideo } from './videoAssembler.js';
import { uploadToYouTube } from './youtubeUploader.js';
import { uploadToTikTok } from './tiktokUploader.js';
import { uploadToInstagram } from './instagramUploader.js';
import { GoogleGenAI } from '@google/genai';
import dotenv from 'dotenv';
dotenv.config();

const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY });
const OUTPUT_DIR = path.resolve('output');
const activeJobs = new Set(); // Prevent duplicate processing of same schedule

export function initScheduler() {
    // Run every minute
    cron.schedule('* * * * *', async () => {
        try {
            await processSchedules();
        } catch (err) {
            console.error('[Scheduler] Top-level error:', err);
        }
    });
    console.log('[Scheduler] Autopilot scheduler initialized');
}

async function processSchedules() {
    const now = new Date();
    
    // Find all active schedules whose next_run_at is <= now
    const { data: dueSchedules, error } = await supabase
        .from('schedules')
        .select('*')
        .eq('is_active', true)
        .lte('next_run_at', now.toISOString())
        .order('next_run_at', { ascending: true })
        .limit(10); // Process max 10 per cycle to avoid overload
    
    if (error || !dueSchedules?.length) return;
    
    for (const schedule of dueSchedules) {
        // Skip if already being processed
        if (activeJobs.has(schedule.id)) continue;
        activeJobs.add(schedule.id);
        
        // Run async without awaiting (fire-and-forget per schedule)
        processOneSchedule(schedule)
            .catch(err => console.error(`[Scheduler] Schedule ${schedule.id} error:`, err))
            .finally(() => activeJobs.delete(schedule.id));
    }
}

async function processOneSchedule(schedule) {
    const { id: scheduleId, user_id: userId, niche, topic_theme, tone,
            style, style_prompt, voice_id, duration, target_platforms,
            generated_angles } = schedule;
    
    console.log(`[Scheduler] Processing schedule ${scheduleId} for user ${userId}`);
    
    // Step 0: Check credits
    const { data: profile } = await supabase
        .from('profiles')
        .select('credits, plan')
        .eq('id', userId)
        .single();
    
    if (!profile || profile.credits <= 0) {
        console.log(`[Scheduler] User ${userId} has no credits. Skipping.`);
        await updateScheduleAfterRun(scheduleId, schedule, {
            last_error: 'Insufficient credits',
            consecutive_failures: schedule.consecutive_failures + 1
        });
        return;
    }
    
    // Step 1: Generate unique angle using Gemini
    const angle = await generateUniqueAngle(topic_theme, niche, generated_angles);
    
    // Step 2: Generate the video using existing pipeline
    const jobId = uuidv4();
    const jobDir = path.join(OUTPUT_DIR, jobId);
    if (!fs.existsSync(jobDir)) fs.mkdirSync(jobDir, { recursive: true });
    
    try {
        // Generate script
        const script = await generateScript({
            topic: angle,
            niche,
            tone,
            duration,
            style: style_prompt
        });
        
        // Generate voiceover
        const { audioPath, timestamps } = await generateVoiceover({
            text: script.cleanScript,
            voiceId: voice_id,
            outputDir: OUTPUT_DIR,
            jobId
        });
        
        // Fetch AI images
        const searchTerms = script.segments.map(s => s.imagePrompt || s.searchTerm);
        const clips = await fetchStockFootage({
            searchTerms,
            outputDir: OUTPUT_DIR,
            jobId,
            orientation: 'portrait'
        });
        
        // Assemble video
        const isFreeTrial = profile.plan === 'free';
        const videoPath = await assembleVideo({
            clips, audioPath, outputDir: OUTPUT_DIR, jobId,
            script, timestamps, watermark: isFreeTrial
        });
        
        // Upload to Supabase Storage
        const videoBuffer = fs.readFileSync(videoPath);
        const storagePath = `videos/${userId}/${jobId}.mp4`;
        
        const { error: uploadError } = await supabase.storage
            .from('reelswave-videos')
            .upload(storagePath, videoBuffer, {
                contentType: 'video/mp4',
                cacheControl: '3600'
            });
        
        if (uploadError) throw new Error(`Storage upload failed: ${uploadError.message}`);
        
        const { data: { publicUrl } } = supabase.storage
            .from('reelswave-videos')
            .getPublicUrl(storagePath);
        
        // Save to videos table
        const { error: dbError } = await supabase
            .from('videos')
            .insert({
                id: jobId,
                user_id: userId,
                title: script.title || angle,
                topic: angle,
                niche,
                tone,
                script: JSON.stringify(script),
                video_url: publicUrl,
                storage_path: storagePath,
                status: 'completed',
                hashtags: script.hashtags || [],
                schedule_id: scheduleId,
                is_autopilot: true
            });
        
        if (dbError) console.error('[Scheduler] DB insert error:', dbError);
        
        // Deduct credit
        await supabase
            .from('profiles')
            .update({ credits: profile.credits - 1 })
            .eq('id', userId);
        
        // Cleanup local files
        try { fs.unlinkSync(videoPath); } catch (e) {}
        
        // Step 3: Upload to target platforms
        await uploadToPlatforms({
            userId, videoId: jobId, scheduleId, videoPath, videoUrl: publicUrl,
            title: script.title || angle,
            description: script.hook || '',
            hashtags: script.hashtags || [],
            targetPlatforms: target_platforms
        });
        
        // Step 4: Update schedule state
        const updatedAngles = [...(generated_angles || []), angle].slice(-50); // Keep last 50
        await updateScheduleAfterRun(scheduleId, schedule, {
            last_error: null,
            consecutive_failures: 0,
            generated_angles: updatedAngles
        });
        
    } catch (err) {
        console.error(`[Scheduler] Video generation failed for schedule ${scheduleId}:`, err);
        await updateScheduleAfterRun(scheduleId, schedule, {
            last_error: err.message,
            consecutive_failures: schedule.consecutive_failures + 1
        });
    }
}

async function uploadToPlatforms({ userId, videoId, scheduleId, videoPath, videoUrl,
                                    title, description, hashtags, targetPlatforms }) {
    for (const platform of targetPlatforms) {
        // Create upload log entry
        const { data: log } = await supabase
            .from('upload_logs')
            .insert({
                user_id: userId,
                video_id: videoId,
                schedule_id: scheduleId,
                platform,
                status: 'uploading',
                title,
                description,
                hashtags
            })
            .select()
            .single();
        
        try {
            let result;
            
            switch (platform) {
                case 'youtube':
                    // YouTube needs the local file path (stream upload)
                    // Re-download from Supabase if local file was cleaned up
                    let localPath = videoPath;
                    if (!fs.existsSync(localPath)) {
                        localPath = await downloadFromSupabase(videoUrl, videoId);
                    }
                    result = await uploadToYouTube({ userId, videoPath: localPath, title, description, tags: hashtags });
                    break;
                    
                case 'tiktok':
                    let tiktokPath = videoPath;
                    if (!fs.existsSync(tiktokPath)) {
                        tiktokPath = await downloadFromSupabase(videoUrl, videoId);
                    }
                    result = await uploadToTikTok({ userId, videoPath: tiktokPath, title, description, tags: hashtags });
                    break;
                    
                case 'instagram':
                    // Instagram uses public URL directly
                    result = await uploadToInstagram({ userId, videoUrl, title, description, tags: hashtags });
                    break;
            }
            
            // Update log as completed
            await supabase
                .from('upload_logs')
                .update({
                    status: 'completed',
                    platform_video_id: result?.platformVideoId,
                    platform_url: result?.platformUrl,
                    completed_at: new Date().toISOString()
                })
                .eq('id', log.id);
                
        } catch (err) {
            console.error(`[Scheduler] Upload to ${platform} failed:`, err);
            await supabase
                .from('upload_logs')
                .update({
                    status: 'failed',
                    error_message: err.message
                })
                .eq('id', log.id);
        }
    }
}

async function downloadFromSupabase(publicUrl, videoId) {
    // Download the video back from Supabase to a temp local path
    const response = await fetch(publicUrl);
    const buffer = Buffer.from(await response.arrayBuffer());
    const tempPath = path.join(OUTPUT_DIR, `${videoId}_redownload.mp4`);
    fs.writeFileSync(tempPath, buffer);
    return tempPath;
}

function computeNextRunUtc(uploadTime, timezone) {
    // uploadTime is like '14:30:00', timezone is like 'America/New_York'
    // Compute next occurrence in UTC
    const now = new Date();
    
    // Use Intl.DateTimeFormat to work with timezone
    const [hours, minutes] = uploadTime.split(':').map(Number);
    
    // Create a date in the user's timezone for today
    const userNow = new Date(now.toLocaleString('en-US', { timeZone: timezone }));
    const target = new Date(userNow);
    target.setHours(hours, minutes, 0, 0);
    
    // If the time already passed today, schedule for tomorrow
    if (target <= userNow) {
        target.setDate(target.getDate() + 1);
    }
    
    // Convert back to UTC
    const offsetMs = target.getTime() - userNow.getTime() + now.getTime() - now.getTime();
    // Simpler: compute the UTC time by determining the offset
    const formatter = new Intl.DateTimeFormat('en-US', {
        timeZone: timezone,
        year: 'numeric', month: '2-digit', day: '2-digit',
        hour: '2-digit', minute: '2-digit', second: '2-digit', hour12: false
    });
    
    // For production, use a library like luxon for reliable timezone math
    // This is a simplified calculation
    const diff = now.getTime() - new Date(formatter.format(now)).getTime();
    const nextRun = new Date(target.getTime() + diff);
    
    return nextRun.toISOString();
}

async function updateScheduleAfterRun(scheduleId, schedule, overrides) {
    const nextRun = computeNextRunUtc(schedule.upload_time, schedule.timezone);
    
    // Auto-pause after 5 consecutive failures
    const shouldPause = (overrides.consecutive_failures || 0) >= 5;
    
    await supabase
        .from('schedules')
        .update({
            last_run_at: new Date().toISOString(),
            next_run_at: nextRun,
            is_active: shouldPause ? false : schedule.is_active,
            ...overrides,
            updated_at: new Date().toISOString()
        })
        .eq('id', scheduleId);
}
```

```javascript
            .from('schedules')
            .select('*')
            .eq('id', req.params.id)
            .eq('user_id', req.user.id)
            .single();
        
        if (!existing) return res.status(404).json({ error: 'Schedule not found' });
        
        const allowedFields = [
            'niche', 'topic_theme', 'tone', 'style', 'style_prompt',
            'voice_id', 'duration', 'upload_time', 'timezone',
            'target_platforms', 'is_active'
        ];
        
        const updates = {};
        for (const field of allowedFields) {
            if (req.body[field] !== undefined) {
                updates[field] = req.body[field];

        // Recompute next_run_at if time or timezone changed

                updates.upload_time || existing.upload_time,
                updates.timezone || existing.timezone

        if (updates.is_active === true && !existing.is_active) {

                updates.timezone || existing.timezone

            updates.consecutive_failures = 0;
            updates.last_error = null;

            .from('schedules')
            .update(updates)
```

## Key Architectural Decisions

1. **Why `node-cron` over a separate worker process?** The existing backend is a single Express process with an in-memory job queue. Adding `node-cron` maintains this simplicity. For production scale, this should eventually move to a dedicated worker (e.g., BullMQ + Redis), but `node-cron` matches the current architecture.

2. **Why `next_run_at` pre-computation?** Rather than parsing timezone + time every minute for every schedule, we pre-compute the next UTC trigger time and store it. The cron job just does a simple `lte()` query.

3. **OAuth state stored in-memory Map.** This matches the existing pattern (in-memory job queue). The state entries should have a TTL cleanup (setTimeout to delete after 10 minutes) to prevent memory leaks.

4. **Scheduler fire-and-forget pattern.** Each due schedule is processed concurrently (not sequentially), but there is a `Set` guard to prevent double-processing if a schedule takes longer than 1 minute.

5. **Instagram uses public URLs.** The video is already uploaded to Supabase Storage with a public URL. Instagram's Graph API requires a publicly accessible video URL, so this works without downloading the file locally.

6. **YouTube and TikTok need local file streams.** Both require uploading the raw binary. The scheduler downloads the video from Supabase back to disk if the local file was already cleaned up.

7. **`generated_angles` JSONB column** prevents topic repetition. The last 50 angles are stored and passed to Gemini as a negative constraint. This ensures daily variety.

8. **Auto-pause after 5 consecutive failures** prevents burning credits on broken pipelines. The user sees the error in the UI and can fix the issue (re-connect account, etc.) before resuming.
